
# Node js JWT/products/users


This project is to test the JWT with nodejs and mongodb, with a simple system of users and products with authentication and validation


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`URL_CONNECT` 
this a url to connect a mongoDB

`SECRET_KEY` 
this is the word for jsonwebtoken which is used to login users

## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
  npm run dev
```
